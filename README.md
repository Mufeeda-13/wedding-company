Wedding Company – Frontend Assignment

This repository contains the completed frontend assignment developed as part of a hiring evaluation. The objective of the project was to demonstrate an understanding of core frontend concepts, clean code practices, component structuring, and the ability to build a functional and visually consistent user interface within a short timeframe.

Project Overview

The project focuses on creating a simple, structured, and easy-to-maintain frontend application. Although minimalistic, the codebase reflects:

Attention to clean architecture

Organized folder hierarchy

Readable and maintainable JavaScript

A clear separation of concerns between files

Efficient use of Vite for rapid development and bundling

This repository showcases the ability to take a requirement and translate it into a working interface using modern frontend tools and development standards.

Features

Responsive layout structured using HTML and CSS

Modular JavaScript architecture enabling future extendability

Fast development environment powered by Vite

Simple and understandable project structure suitable for demonstration and review

Code written with clarity and maintainability in mind

Technologies Used

HTML5 for markup

CSS for styling and layout

JavaScript (ES6+) for functionality

Vite as the build and development tool

These technologies were chosen to ensure a lightweight and efficient development workflow, while maintaining clarity and simplicity in the final codebase.

How to Run the Project

To view or test the project locally, follow the steps below.

Step 1: Install Dependencies

Open the project directory in a terminal and run:

npm install

Step 2: Start the Development Server

Run the following command:

npm run dev

Step 3: Open in Browser

The project will be available at:

http://localhost:5173/


This setup uses Vite, which provides fast hot-reloading and an optimized development environment.
